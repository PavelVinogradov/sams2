Сохранение конфигурации SAMS и backup базы логов SQUID

Сохранение конфигурации SAMS

Вы можете сохранить настройки SAMS:
  • используя утилиту mysqldump
    mysqldump -u root squidctrl
  • из WEB интерфейса SAMS
    В дереве конфигурации и управления WEB интерфейса SAMS выберите пункт BackUP (сохранение конфигурации). Далее в нижнем фрейме правой панели нажмите кнопку "Сохранить настройки в файл". Далее нажмите кнопку "Сохранить", Далее нажмите на появившуюся ссылку "Загрузить файл".
    Вам будет предложено сохранить на диск файл с именем подобным:
    samsdb-11Jul2005-19-42-09.sql.gz

    BACKUP базы логов SQUID

    Вы можете сохранить базы логов SQUID:
  • используя утилиту mysqldump
    mysqldump -u root squidlog
  • из WEB интерфейса SAMS
    В дереве конфигурации и управления WEB интерфейса SAMS выберите пункт SQUID. Далее в нижнем фрейме правой панели нажмите кнопку "Сохранить логи SQUID в архиве". Далее в открывшемся диалоге выберите период времени, за который вы хотите сохранить данные, нажмите кнопку "Ок", Далее нажмите на появившуюся ссылку "Загрузить файл".
    Вам будет предложено сохранить на диск файл с именем подобным:
    squidlogdump-11Jul2005-19-47-04.sql.gz
    Внимание!!!Сохраняемые данные при этой операции из базы не удаляются. Для удаления нажмите кнопку "Очистить базу логов SQUID" . Далее вам предложат выбрать период времени, данные за который необходимо удалить из базы.

    Автоматический BACKUP базы логов SQUID

    Вы можете настроить автоматическое сохранение базы логов SQUID. В настройках SAMS выберите количество месяцев, данные за которые будут сохраняться в базе SAMS.
    Первого числа нового месяца старые данные будут сохраняться на диск и удаляться из базы данных. Данные сохраняются в виде текстового файла с SQL запросами. Файлы доступны из web интерфейса SAMS (раздел SQUID) и хранятся в каталоге:
    /web_interface_path/backup

    Восстановление конфигурации SAMS

    Вы можете восстановить сохраненные ранее настройки SAMS:
  • из WEB интерфейса SAMS
    В дереве конфигурации и управления WEB интерфейса SAMS выберите пункт BackUP (сохранение конфигурации). Далее в нижнем фрейме правой панели нажмите кнопку "Восстановить настройки из файла". Далее в открывшемся диалоге выберите файл с сохраненными настройками и нажмите кнопку "Загрузить файл", Далее в открывшемся диалоге выберите пункты, которые вы хоти восстановить:
  • Группы пользователей
  • Пользователи
  • Список перенаправления запроса и запрета доступа
  • Шаблоны пользователей
    Внимание!!! Старые данные из выбранных пунктов будут при этом уничтожены!

    Загрузка сохраненных данных в базу логов SQUID

    Вы можете загрузить сохраненные ранее базы логов SQUID:
  • из WEB интерфейса SAMS
    В дереве конфигурации и управления WEB интерфейса SAMS выберите пункт SQUID. Далее в нижнем фрейме правой панели нажмите кнопку "Загрузить в базу логи SQUID". Далее в открывшемся диалоге выберите файл с сохраненными настройками и нажмите кнопку "Загрузить файл"
