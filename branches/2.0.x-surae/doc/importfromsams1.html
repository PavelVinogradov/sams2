5.2.2.1 Импорт настроек и пользователей из SAMS 1.X

Web интерфейс SAMS2 позволяет импортировать данные из базы SAMS 1.x. Обеспечивается импорт:

  • Групп пользователей
  • Временных диапазонов
  • Шаблонов
  • Пользователей
  • Списков URL
  • Настроек прокси сервера

Нажмите кнопку "Импорт настроек из базы данных SAMS 1.x". Откроется диалог настройки подключения к базе данных SAMS и выбора данных для импорта.


Рисунок 5.2.2.2. Диалог настройки подключения к базе данных SAMS и выбора данных для импорта

Настройки подключения к базе данных SAMS1

  • DB Hostname - сервер на катором установлен MySQL.
    localhost - если база данных установлена на этом же сервере.
    hostname:port - если база данных установлена на другом сервере.
  • DB Login - логин, под которым будет установлено соединение с базой SAMS 1.x
  • DB Password - пароль

Настройки импорта данных

  • Импортировать пользователей, группы пользователей и шаблоны из SAMS версии 1.0.x - при активации этой функции производится импорт пользователей, групп пользователей и шаблонов из SAMS 1.x
  • импортировать имена пользователей - функция доступна при активной функции импорта пользователей из SAMS 1.x.
    • без изменения
    • преобразовывать в строчные буквы
    • преобразовывать в ЗАГЛАВНЫЕ буквы
      преобразование логинов пользователей используется при использовании NTLM авторизации пользователей, если, например, на старом сервере wbinfo отдавал логины пользователей заглавными буквами, а на новом - строчными.
  • Импортировать списки запрета доступа из SAMS версии 1.0.x - В SAMS2 будут импортиоваы списки запрета доступа из SAMS 1.
  • Создать прокси сервер и импортировать настройки - В SAMS2 будет создан новый прокси сервер с настройками, импортированными из SAMS1.

5.2.2.2 Импорт данных о трафике пользователей из SAMS 1.X

Импорт данных из SAMS 1.x предназначен для более легкого перевода прокси сервера SQUID под управление SAMS2. Этой функцией обеспечивается импорт данных о трафике пользователей.

Нажмите кнопку "Импорт данных о трафике пользователей из базы данных SAMS 1.x". Откроется диалог настройки подключения к базе данных SAMS и выбора данных для импорта.


Рисунок 5.2.2.4. Диалог настройки подключения к базе данных SAMS

Настройки подключения к базе данных SAMS1

  • DB Hostname - сервер на катором установлен MySQL.
    localhost - если база данных установлена на этом же сервере.
    hostname:port - если база данных установлена на другом сервере.
  • DB Login - логин, под которым будет установлено соединение с базой SAMS 1.x
  • DB Password - пароль

Далее нажимаем кнопку "Импортировать". Веб интерфейсом будет произведен поиск данных о трафике в базе данных SAMS 1.x и выведен диалог выбора диапазона дат для импорта.


Рисунок 5.2.2.4. Диалог выбора диапазона загрузки данных о трафике пользователей.

Выберите диапазон и нажмите кнопку "Ok".

Если происходит импорт данных в СУБД PostgreSQL и данные о трафике в базе SAMS 1.x хранятся в кодировке KOI8-R то могут быть проблемы с импортом. В этом случае импортируйте данные небольшими диапазонами по несколько дней.
