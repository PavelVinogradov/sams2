Журнал событий SAMS

Журнал событий SAMS предназначен для хранения записей о сыбытиях, просходящих в процессе функционировани SAMS.

Для просмотра событий в дереве конфигурации и управления WEB интерфейса SAMS выберите пункт "Журнал"

Регистрируемые события

  • - события, связанные с созданием, удалением, активацией пользователей
  • - события, связанные с созданием, удалением групп пользователей
  • - события, связанные с очисткой базы логов SQUID
  • - события, происходящие в процессе функционирования программ samsdaemon, sams, samsf

    Уровень детализации событий выставляется в настройках SAMS.

    Для этого в дереве конфигурации и управления WEB интерфейса SAMS выберите пункт "Администрирование SAMS" -> Настройки SAMS.

    Чем выше уровень детализации логов, тем больше они занимают места в базе данных SAMS! Выставлять большие значения детализации логов рекомендую только на период установки и отладки.
