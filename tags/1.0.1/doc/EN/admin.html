The SAMS users with Administrator rights

This page contained information about the SAMS users with Administrator rights.
The SAMS users
Access to the SAMS WEB-interface can have a users, registered in the SAMS.

The user with Administrator rights
In the SAMS default included user Admin. This user can be set setttings of SAMS and seen statistic reports all users of SAMS.

The user "Auditor"
In the SAMS default included user Auditor. This user can be seen statistic reports all users of SAMS.

The SAMS users
The users, registered in the SAMS, can be seen own statistic reports.

Использование графического интерфейса
Для доступа к окну управления пользователями, имеющими права администратора системы, через удаленный интерфейс необходимо:
