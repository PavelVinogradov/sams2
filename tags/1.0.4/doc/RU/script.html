Скрипты, выполняемые при отключении пользователя

Утилиты sams и samsf имеют возможность выполнять скрипт при отключении пользователя.
В состав sams входят 2 скрипта:
  • send_email_message - посылает сообщение администратору по электронной почте
  • send_winpopup_message - посылает сообщение администратору через winpopup сообщение

    Вы можете добавить любой свой скрипт.
    Скрипты должны находиться в директории
    /sams_web_interface/src/scripts

    Вы можете написать собственный скрипт для отправки сообщений, например, на сотовый телефон.

    В скрипте используются переменные:
    $SAMSUSERNAME - имя пользователя SAMS, формируется утилитами sams, samsf
    $ADMINADDR - адрес администратора, заносится при настройке sams.
