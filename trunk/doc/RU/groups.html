Группы пользователей

Группы пользователей позволяют организовать группировку пользователей в WEB интерфейсе для удобства администрирования.

Создание новой группы пользователей

Для создания новой группы пользователей в дереве конфигурации и управления WEB интерфейса SAMS выберите пункт "Группы пользователей". Далее в нижнем фрейме правой панели нажмите кнопку "Добавить группу пользователей".
В открывшемся диалоге введите название группы пользователей.

Получение списка групп пользователей с сервера AD

Данный режим работает, если в качестве авторизации пользователя на сервере SQUID выбран режим "Active Directory".

Для получения списка групп пользователей с сервера AD в дереве конфигурации и управления WEB интерфейса SAMS выберите пункт "Группы пользователей". Далее в нижнем фрейме правой панели нажмите кнопку "Загрузка групп пользователей из домена Active Directory".

В открывшемся диалоге выберите группы, которые нужно внести в WEB интерфейс SAMS и нажмите кнопку "Добавить".
Для подключения и авторизации на сервере AD необходимо в файле sams.conf внести настройки:

LDAPSERVER=192.168.0.11 - имя сервера AD или его IP адрес
LDAPBASEDN=security.local - имя домена AD
LDAPUSER=username - имя пользователя, входящего в группу "администраторы домена"
LDAPUSERPASSWD=passwd - пароль
LDAPUSERSGROUP=Users
от имени этого пользователя будет производиться проверка авторизации пользователей в WEB интерфейсе SAMS и запрос списков пользователей и групп в AD.
