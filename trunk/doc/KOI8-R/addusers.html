Запуск системы в эксплуатацию:

Установите ПО ( samba 2.x или samba 3.x, squid, apache, mysql, sams) согласно документации, и попробуйте загрузить графический интерфейс SAMS. Если интерфейс пользователя недоступен или работает с ошибками, прочитайте здесь.

Введите пароль администратора (пользователь admin, пароль - смотрите в архиве, файл INSTALL.RUS)

Настройте WEB интерфейс:

Настройки WEB интерфейса -> астройки WEB интерфейса

Выберите:

  • Язык WEB интерфейса
  • Размер единицы трафика вашего провайдера

    Разрешите доступ пользователей к своей статистике.
    Если вы хотите, чтобы пользователи имели доступ к своей статистике, поставьте галочки около нужных пунктов:

  • по трафику за день
    • - показывает объем скаченного пользователем трафика по дням
  • по посещенным URL
    • - показывает пользователю список посещенных им URL

    Настройте SAMS:

    Администрирование SAMS -> Настройки SAMS

    Смена пароля администратора:

    Администрирование SAMS -> Сменить пароль пользователя

    Создайте шаблоны пользователей.

    Создайте шаблоны пользователей. Шаблоны включают в себя списки запрета доступа, перенаправления запроса и скорости канала пользователей.

    Создайте группы пользователей.

    Создайте группы пользователей. Группы не несут никакой смысловой нагрузки и не привязываются, например к группам в домене, они созданы только для удобства работы с пользователями в интерфейсе SAMS.

    Зарегистрируйте пользователей в системе:

    Пункт "Пользователи".

  • Если у вас используется NTLM или AD аутентификация (в домене Windows), вы можете вводить пользователей из формы регистрации по-одному, или ввести сразу группу пользователей из контроллера домена:
    Пользователи -> Загрузить всех пользователей домена
    Выберите пользователей из списка, определите их в группу и задайте им шаблон.

  • Если у вас используется доступ к SQUID по ip адресам, вы можете вводит пользователей из формы регистрации. Необходимо задать имя пользователя, ip адрес и пароль для доступа пользователей к их статистике

  • Если у вас используется NCSA аутентификация, вы можете вводить пользователей из формы регистрации. Необходимо задать имя пользователя и пароль для доступа пользователей к их статистике

    Перезагрузите SQUID

    Для внесения изменений в настройки SQUID перезапустите SQUID
    Squid -> Реконфигурирование SQUID
