2 Список дополнительных пакетов

2.1 Обязательные пакеты

Несмотря на то что пакеты mysql, postgresql и unixODBC внесены в раздел необязательных, хотя-бы один из этих пакетов должен быть установлен, иначе не будет возможности использовать ни одну СУБД, а значит SAMS будет неработоспособен.

Модуль SAMS2 Наименование пакета Описание пакета
sams2 squid Кэширующий прокси-сервер Squid


sams2-web httpd HTTP сервер Apache
php Встраиваемый в HTML cкриптовый язык PHP
mod_php Встроенный интерпретатор PHP для Веб-сервера Apache
gd Графическая библиотека для быстрого создания изображений формата PNG или JPEG
php-gd Модуль для приложений PHP, использующих графическую библиотеку gd

2.2 Необязательные пакеты

Модуль SAMS2 Наименование пакета Описание пакета Зачем может понадобится
sams2 pcre Совместимая с Perl библиотека регулярных выражений Для использования регулярных выражений при определении различных фильтров.
mysql Клиент и разделяемые библиотеки MySQL Для использования MySQL в качестве СУБД для работы SAMS2.
postgresql Клиентские программы и библиотеки для PostgreSQL Для использования PostgreSQL в качестве СУБД для работы SAMS2.
unixODBC Менеджер всех драйверов ODBC для Linux Если планируется использование ODBC для подключения к СУБД, не поддерживаемой в SAMS2. Не забывайте что еще понадобится драйвер ODBC для этой СУБД.
doxygen Система создания документации для C/C++ Если необходима документация по исходным кодам.
sams2-web php-ldap Модуль для приложений PHP, использующих LDAP Для авторизации в ActiveDirectory или OpenLDAP.
fpdf
Для экспорта отчетов в pdf файл с использованием fpdf.
pdflib
Для экспорта отчетов в pdf файл с использованием pdflib.

2.3 Пакеты, необходимые при установке из исходных текстов

Этот список является дополнением к обязательному и необязательному спискам.

Модуль SAMS2 Наименование пакета Описание пакета Зачем может понадобится
--- autoconf Утилита GNU для автоматической конфигурации исходных текстов Для автоматической конфигурации исходных текстов.
automake Утилита GNU для автоматического создания файлов Makefile Для автоматического создания файлов Makefile.
libtool Средство GNU для создания переносимых библиотек Для создания переносимых библиотек.
sams2 pcre-devel Файлы разработки для pcre Для использования регулярных выражений при определении различных фильтров.
mysql-devel Файлы для разработки приложений MySQL Для использования MySQL в качестве СУБД для работы SAMS2.
libstdc++-devel Файлы заголовков и библиотеки для разработки на C++ Для компиляции исходных кодов.
gcc-c++ Поддержка C++ для GCC Для компиляции исходных кодов.
postgresql-devel Библиотеки и файлы заголовков PostgreSQL, необходимые для разработки приложений Для использования PostgreSQL в качестве СУБД для работы SAMS2.
unixODBC-devel Файлы разработчика для программ, использующих библиотеку unixODBC Если планируется использование ODBC для подключения к СУБД, не поддерживаемой в SAMS2. Не забывайте что еще понадобится драйвер ODBC для этой СУБД.
