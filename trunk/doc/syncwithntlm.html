5.4.3 Синхронизация с доменом Windows NT 4.0

SAMS позволяет синхронизировать пользователей шаблонов SAMS с пользователями групп домена Windows NT 4.0. Синхронизация пользователей производится для шаблонов с именами групп, импортированных из домена Windows NT 4.0.

Настройка подключения WEB интерфейса SAMS к домену Windows NT 4.0


Рис 7.2.1. Настройка подключения WEB интерфейса SAMS к домену Windows NT 4.0

Параметры, задаваемые для настройки доступа к серверу домена Windows NT 4.0:

Сервер домена Windows IP адрес контроллера домена Windows NT 4.0
Домен Windows Домен Windows NT 4.0
Администратор домена Windows пользователь, от имени которого WEB интерфейс SAMS будет подключаться к домену Windows NT 4.0
Пароль администратора домена Windows пароль
Группа пользователей домена Windows Группа домена Windows NT 4.0, которая содержит пользователей

После сохранения параметров можно протестировать подключение к домену Windows NT 4.0 нажав кнопку "test connections to Windows NT 4.0 domain"

Импорт пользователей групп домена Windows NT 4.0 в SAMS

Импорт производится из WEB интерфейса SAMS.


Рис 7.2.2. Импорт групп пользователей из домена Windows NT 4.0

При импорте создаются шаблоны SAMS c названиями групп домена Windows NT 4.0. Импорт пользователей производится в группы SAMS:

  • существующую группу SAMS
  • во вновь создаваемую группу SAMS
  • во вновь создаваемую группу SAMS с именем группы AD

В форме импорта пользователей из домена Windows NT 4.0 задаются параметры:

  • Объем трафика пользователя шаблона по умолчанию
  • Скорость канала для всего шаблона
  • Скорость канала для отдельного пользователя
  • Период лимита трафика
    Далее эти параметры будут присвоены шаблонам пользователей.

    Вывод шаблона SAMS из синхронизации с доменом Windows NT 4.0

    Если необходимо вывести шаблон SAMS из синхронизации с доменом Windows NT 4.0, необходимо выбрать шаблон из списка и нажать кнопку "Удалить выбранные шаблоныы из синхронизации с доменом Windows"

    Ввод шаблона SAMS в синхронизацию с доменом Windows

    Если необходимо вернуть шаблон SAMS в синхронизацию с доменом Windows, необходимо выбрать шаблон из списка шаблонов SAMS, и нажать кнопку "Добавить выбранные шаблоны в синхронизацию с доменом Windows"

    Синхронизация пользователей с доменом Windows

    Синхронизация производится скриптом syncfromntlm. Скрипт вызывается из cron:
    */15 * * * * root cd /usr/local/share/sams2; php bin/syncfromntlm
