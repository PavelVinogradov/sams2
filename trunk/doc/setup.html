4 Настройка

  1. Отредактируйте конфигурационный файл sams2.conf, согласно вашей конфигурации.
  2. В любом web браузере откройте страницу http://yourservername/sams2. Введите имя и пароль администратора базы данных, установите флаг "Create SAMS DB user" и введите имя и пароль пользователя SAMS2 для последующего подключения к БД (тот, который указали в sams2.conf). На данном этапе этого пользователя не должно существовать в БД. (Не удаляйте @localhost в имени пользователя!)
  3. Запустите демон. 
    service sams2 start



  4. После нажатия кнопки "Starting SAMS webinterface", можно переходить к этапу администрирования.
