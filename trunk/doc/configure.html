5 Администрирование

5.1.1 Первый запуск веб интерфейса SАMS. Создание базы данных SAMS.

В любом web браузере откройте страницу http://yourservername/sams2. Если веб интерфейс не смог присоединиться к базе данных, Появится окно, предлагающее запустить визард, создающий базу данных.


Рис 5.1.1. Первый запуск WEB интерфейса SAMS. База данных SAMS не обнаружена.


Рис 5.1.2. Визард создания базы данных SAMS. Диалог выбора кодировки веб интерфейса.

Далее производится проверка параметров настроек php, необходимых для функционирования веб интерфейса SAMS.


Рис 5.1.3. Визард создания базы данных SAMS. Проверка настроек php.


Рис 5.1.5. Первый запуск WEB интерфейса SAMS. Диалог подключения к базе данных SAMS.

Внимание!!! При использовании MySQL база данных создается скриптом автоматически, при использовании PostgreSQL ее необходимо создать заранее.


Рис 5.1.6. Первый запуск WEB интерфейса SAMS. Лог создания базы данных SAMS.

Далее на экран будет выведен лог создания базы данных SAMS. Если все прошло удачно, нажмите на кнопку "Start SAMS web interface"

5.1.2 Запуск веб интерфейса SАMS.

В любом web браузере откройте страницу http://yourservername/sams2. Появится окно, изображенное ниже. нажмите на ссылку "SAMS management interface" и введите логин и пароль администратора. Значения по-умолчанию admin и qwerty.
