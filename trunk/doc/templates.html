5.7 Шаблоны пользователей

Временные диапазоны используются в шаблонах пользователей для задания времени, когда пользователи могут получить доступ к прокси-серверу SQUID.

Добавление шаблона пользователей в WEB интерфейсе SAMS


Рис 5.7.1. Добавление шаблона пользователей в WEB интерфейсе SAMS

  • Объем трафика пользователя шаблона по умолчанию - объем трафика, назначаемый при добавлении новому пользователю
  • Скорость канала для всего шаблона - максимальная скорость канала для всех пользователей шаблона
  • Скорость канала для отдельного пользователя - максимальная скорость канала для одного польззователя шаблона
  • Способ авторизации пользователей - способ авторизации пользователей на прокси сервере SQUID
  • Период лимита трафика - период действия ограничения трафика пользователей.
    можно задать периоды ограничения трафика:
    Месяц
    День
    Другой период - период в днях, задаваемый администратором
    Дата следующего обнуления счетчика трафика - дата, когда произойдет обнуление счетчиков трафика. Действует при заданном периоде ограничения трафика "Другой период". При периодах "Месяц" или "Неделя" значение не задано.
  • Временные диапазоны - диапазоны времени, во время которых пользователи могут подключаться к прокси серверу. Шаблон может включать в себя несколько временных диапазонов

Настройка свойств шаблона пользователей

WEB интерфейс SAMS позволяет изменять параметры шаблонов пользователей.


Рис 5.7.2. Изменение параметров шаблона пользователей

В этом диалоге вы можете задать списки URL:

  • Списки SAMS - списки перенаправления и запрета доступа
  • Доступ запрещен ко всем URL - доступ запрещен ко всем URL, кроме разрешенных в списках "Доступ к URL разрешен"
  • Доступ к URL разрешен - списки URL, доступ к которым всегда разрешен
