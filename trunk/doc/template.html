5.7 Шаблоны пользователей

Временные диапазоны используются в шаблонах пользователей для задания времени, когда пользователи могут получить доступ к прокси-серверу SQUID.

Добавление шаблона пользователей в WEB интерфейсе SAMS


Рис 5.7.1. Добавление шаблона пользователей в WEB интерфейсе SAMS

  • Название - название временного диапазона
  • Дни недели - дни недели, в которые пользователи смогут получить доступ к прокси-серверу SQUID
  • Время - время, в которые пользователи смогут получить доступ к прокси-серверу SQUID

Шаблон может включать в себя несколько временных диапазонов

Настройка свойств временного диапазона

WEB интерфейс SAMS позволяет изменять параметры временного диапазона.
