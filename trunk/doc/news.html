SAMS2 Приложения

Оглавление

Приложение 4. Новое в версии 2

Общее

  1. Полностью переписаны демоны. Теперь используются C++ и STL.
  2. Использование jQuery в web интерфейсе.
  3. Прямая поддержка MySQL и PostgeSQL, а также любой другой СУБД, используя ODBC.
  4. Корректная работа с несколькими прокси серверами.
  5. Возможность автоматического добавления пользователей.
  6. Авторизация в LDAP (OpenLDAP, Fedora Directory Server, ...).
  7. Использование плагинов для вывода информации о прокси сервере.
  8. Вторичный шаблон пользователя, используемый при превышении ограничений основного.
  9. Индивидуальный лимит трафика пользователя.
  10. Несколько временных интервалов для одного шаблона.
